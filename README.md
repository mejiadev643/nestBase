## Description

[TOTO]Api made with prisma and  nestjs

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Some impotants commands!!

``` bash
#Create a model
$ npm run createModel ModelName

#merge Schema
$  npm run mergeSchema

#create migrations
$ npx prisma migrate dev --name MigrationName

#run seeders
$ npm run seeder
```
